Detailed Description

: Source file for mpbAsSwitch_STM32 library classes methods

The library builds several switch mechanisms replacements out of simple push buttons or similar equivalent digital signal inputs. By using just a push button (a.k.a. momentary switches or momentary buttons, MPB for short from here on) the classes implemented in this library will manage, calculate and update different parameters to generate the embedded behavior of standard electromechanical switches.
